`Rollup`les champs sont utilisés pour agréger les données des champs de la table associée. Souvent utilisé pour calculer des totaux, des moyennes et d’autres données globales.

## Créer un champ de cumul

1. Cliquer sur`+`icône à droite de`Fields header`
2. Dans la liste déroulante modale, entrez le nom du champ (facultatif).
3. Sélectionnez le type de champ comme`Lookup`dans la liste déroulante.
4. Sélectionnez le champ de lien dans la liste déroulante. Il s'agit du champ qui relie la table actuelle à la table associée.
5. Sélectionnez le champ à afficher dans la liste déroulante. C'est le champ qui sera affiché dans le tableau courant.
6. Sélectionnez la fonction d'agrégation dans la liste déroulante. C'est la fonction qui sera utilisée pour agréger les données.
7. Cliquer sur`Save Field`bouton.

![image](/img/v2/fields/types/rollup.png)

### Fonctions d'agrégation

Voici un tableau avec de brèves descriptions pour chacune des fonctions d'agrégation prises en charge par NocoDB :

| Fonction d'agrégation | Descriptif |
|----------------------|------------------------------- -------------------------------|
| Comte | Compte le nombre d'enregistrements dans un ensemble de données. |
| Minimum | Récupère la valeur minimale d'un ensemble de données. |
| Maximale | Récupère la valeur maximale d'un ensemble de données. |
| Moyenne | Calcule la valeur moyenne dans un ensemble de données. |
| Somme | Additionne toutes les valeurs d'un ensemble de données. |
| Comte distinct | Compte le nombre de valeurs distinctes dans un ensemble de données. |
| Somme Distinct | Additionne toutes les valeurs distinctes d'un ensemble de données. |
| Moyenne Distincte | Calcule la moyenne de valeurs distinctes dans un ensemble de données. |
